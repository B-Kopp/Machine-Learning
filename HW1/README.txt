The code for this assignment is stored at: https://github.com/B-Kopp/CS-7641-HW1
There are 10 files, 1 for each model and each dataset. The are named in the format
`CS 7641 HW1 <Model Name> <Dataset>.ipynb

To run the code you need to install Anaconda Jupyter Notebook or parse out the python code into a script.
It is written in Python 3.6

You also need to install the following modules:
xgboost `conda install -c conda-forge xgboost`
numpy `conda install -c anaconda numpy`
pandas `conda install -c anaconda pandas`
scikit-learn `conda install scikit-learn`
time `conda install -c conda-forge time`
matplotlib `conda install -c conda-forge matplotlib`
tensorflow `conda install -c conda-forge tensorflow`
